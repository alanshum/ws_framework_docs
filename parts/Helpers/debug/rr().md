**Description**
Print arguments using `print_r()` for unlimited number of arguments with proper styles (`<pre>`).

--------
**Parameters**
name	type	def_value	desc
arg1, ...	mixed		Arguments to be printed.

--------
**Return Values**
type	desc
void

--------
**Examples**

```php
rr($_SERVER, $P);
```
